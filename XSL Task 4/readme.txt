Задание 4. Генерация меню.

Реализован XSL шаблон для генерации меню.

В качестве входного параметра, шаблон получает nodeset в котором содериться информация о меню. Пункты меню имеющие атрибут is-current=«true», в HTML будут помечены красным цветом.

Если входной параметр не задан, то шаблон получает все данные из XML. XML имеет отличную от входного nodeset структуру. С помощью временных деревьев преобразуем XML к форме с котором умеем работать и формируем меню.

Параметры для запуска xsltproc:
xsltproc --output menu.htm  menu.xsl menu.xml
